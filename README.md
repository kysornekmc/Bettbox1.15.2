<h1 align="center">⚡ Bettbox</h1>
<p align="center">
  <strong>Another Better Mihomo Client</strong>
</p>

Bettbox 是一款使用Mihomo内核、基于 FlClash二次开发的多平台代理客户端。

它秉承“Better Experience更优体验”的原则，在继承原版优秀界面的基础上，深度优化了诸多细节与预置配置，致力于成为性能更强、占用更低、体验更加细腻的 Mihomo 客户端。

---

## 🚀 核心特性

### 💎 深度体验优化
* **开箱即用**：自动化的权限处理与舒适的TUN&VPN体验，预置更加适合中国用户的细节配置，减少繁琐的手动调试。
* **精雕细琢**：重新审视打磨每一处细节，旨在提供更加流畅、更加易用的Mihomo客户端操作体验。

### 🛡️ 安全与稳定性
* **安全校验**：遵循 Mihomo 官方安全建议，并且有着严格的安全校验以及权限控制，有效防范潜在的TOCTOU或非法访问，同时增加了针对配置文件改动错误的优雅回退机制。
* **高稳定性**：通过了使用者和群友长期高强度的压力测试与后台测试，优化了多处极端场景下的使用稳定性问题。

### 🎨 自由可定制化
* **可视化管理**：更加全面且易用的内核配置界面，绝大部分参数支持可视化操作和实时调整。
* **实用小组件**：提供多种美观实用风格的 Widgets，在首页可以掌控全局流量和当前运行状态。
* **个性化定制**：丰富的色彩主题、自定义图标/首页标题与界面布局调整，让每个人都可以拥有自己独一无二的Bettbox。

### 💻 多个平台支持
* **广泛兼容**：支持主流架构，为现代设备额外优化的性能(例如桌面端CPU分级和Android 16K对齐)，同时针对多平台 Desktop ARM64 设备提供了原生适配。
* **社区包容性**：我们倾听社区用户的想法并且会认真评估，你的声音不会无故被淹没和被无视。
* **旧设备关怀**：提供针对旧版本系统和硬件的兼容性版本，确保长周期的使用寿命。

### 🌈 开源纯净透明
* **全自动 CI/CD**：基于 GitHub Actions 的透明构建流程，代码即产物，所见即所得。
* **纯净无广告**：免费，且完全开源，代码接受全方位审计，无需担心额外的隐私问题。
